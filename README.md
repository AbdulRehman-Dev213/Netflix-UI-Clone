# Netflix UI Clone 🎬

A fully responsive Netflix-inspired landing page built using **HTML** and **CSS**, focused on recreating a modern streaming platform interface with clean layouts, interactive elements, and responsive design principles.

## Overview

This project was built to strengthen my frontend development skills by replicating the UI of Netflix and understanding how real-world applications structure their layouts and responsiveness.

It focuses on modern styling techniques, adaptive layouts, and user interaction improvements.

## Features

* Fully responsive design for desktop and mobile devices
* Modern Netflix-inspired UI
* Responsive typography using **CSS ****`clamp()`**
* Flexbox-based layouts for navigation, hero sections, and forms
* CSS Grid implementation for structured content sections
* Interactive hover effects on buttons, FAQ sections, and movie cards
* Floating label animation for email input fields
* Image scaling effects on hover
* Custom section separation using **linear** and **radial gradients**
* Mobile layout adaptation for screens under **900px**

## Technologies Used

* **HTML5**
* **CSS3**
